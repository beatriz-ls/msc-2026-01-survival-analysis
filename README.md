# Survival Analysis — MSc Statistics (2026/01)

Repository for the Survival Analysis course from the MSc Program in Statistics at the Federal University of Minas Gerais (UFMG).

## Course Information

- Program: MSc in Statistics
- Institution: Federal University of Minas Gerais (UFMG)
- Department: Department of Statistics (DEST/UFMG)
- Instructor: Prof. Fabio Demarqui
- Semester: 2026/01

## About this repository

This repository contains:

- Implementations and applications of survival analysis models;
- Codes developed from examples and datasets presented in Colosimo's textbook;
- Solutions to course assignments and exercise lists;
- Computational studies and experiments related to survival analysis.

The materials available here are intended for academic and educational purposes only.
